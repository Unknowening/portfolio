Movie Store Project

Online platform for checking out, reviewing, and buying movies.

Django Python SQLite
Code Live demo

Description

Movies Store presents a comprehensive array of functions for both users and admin. A general overview of its screens and features is written below.

User Accounts

  • User registration
  • Secure user login
  • View order history

Movie Browsing

  • View list of available movies
  • Search movies by title
  • View detailed movie information
  • Read reviews from other users

Shopping Cart and Orders

  • Access shopping cart
  • Add one or more movie items to cart
  • Remove all items from cart
  • Place movie orders
  • View list of past orders

Reviews

  • Create movie reviews
  • Edit personal reviews
  • Delete personal reviews
  • View reviews from other users
  • Report inappropriate reviews

Administrator

Administrators can manage platform data by creating, viewing, updating, or deleting all of the following:
  • users
  • movies
  • reviews
  • orders

System

  • Accessible via browser
  • Supports multiple screen sizes and devices

Process Description

For this project, the process was fairly simple. Since it was mostly a guided project, my workflow mainly involved following the textbook chapters step by step. However, even with a clear path, there was a lot of content to navigate, and I encountered several errors along the way. To address these issues, I first examined the error codes for clues, then retraced my steps and reviewed recent changes to identify anything that might be causing problems. When I was stuck, I also used online resources for additional help. By using these strategies, I was able to develop the application relatively smoothly.
